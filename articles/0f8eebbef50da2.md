# Manusでツールを作成する体験
![](/images/poplecdn.png)
Manusを使って開発効率を上げるために、CDNギャラリーを作成しました。
Xでたまたま見かけた「Pople」という日本人の方が作成したフリーイラストです。

# CDNを選んだ理由
「CDN」形式で誰もが利用できるようにしました。
この形式を選んだ理由は、毎回ダウンロードせずに画像を表示したかったからです。
サーバー上に画像をアップロードしておき、そちらのURLを使うことで開発時に画像を表示したいけど、アップロードの手間を省略できます。
これからかわいいブログを作成したかったので、Popleはまさに僕が求めていたフリーイラストでした！
https://grokipedia.com/page/Content_delivery_network

# 公開形式
しかし、Popleの利用規約には2次配布を禁じているため、個人利用として自分しか利用できないURLを使うことにしました。
（Manusで公開URLの設定ができます）

パターン1：データ収集
「popleのイラストを3つダウンロードして、apiから画像を利用できるようにしてください。」
